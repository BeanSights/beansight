# **BeanSight-project**

## 💻 프로젝트 소개

- 맞춤형 카페 추천 서비스 입니다!
- 지도에서 검색을 통해 원하는 카페를 찾을 수 있습니다.
- 자신의 위치에서 가까운 카페를 찾을 수 있습니다.
- 자신이 원하는 조건에 맞는 카페를 찾을 수 있습니다.
- 리뷰가 많은 검증된 카페를 찾을 수 있습니다.
